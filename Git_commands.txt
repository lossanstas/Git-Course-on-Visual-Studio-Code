git init - инициализация и регистрация рабоыей директории для Git
git status -  проверка состояния файлов и регистрации изменений файлов и папок.
git add readme.txt - регистрация изменений в файле readme.txt 
git add . - регистрация всех файлов, которые были изменены в процессе работы.
git branch test - создание ветки "test". Название может быть любое.
git branch -b test - создание ветки "test" и одновременное перемещение в неё.
git checkout master - перемещение в ветку "master"
git branch -D test - удаление ветки "test"
.gitignore - создание папки, которая не будет фиксироваться для изменений содержания всех файлов, которые там есть. 
Обязательно проиндексировать.
git commit -m "описание" - закрепление именённых файлов в Git. В кавычках кратко описать тему изменений.
git merge readme - синхронизация веток. если находиться в ветке "master" то синхронизируемся с веткой "readme" и наоборот.
Важно точно знать в какой ветке находишься.
git config --global user.name - регистрация на GitHub. Будет использоваться то имя, которое указано в git(Shabunin Stanislav) 
и проверка тех данных, которые уже зарегистрированы.
git config --global user.name "lossanstas" - редактирование имени пользователя в GitHub
git config --global user.email - проверка зарегистрированного адреса электронной почты.
git config --global user.email "abc@def.com" - изменение зарегистрированного адреса или регистрация адреса электронной почты.
git remote add origin git@github.com:lossanstas/Git-Course-on-Visual-Studio-Code.git - регистрация данных на локальном
компьютере с репозиторием на GitHub-е
git branch -M main - переименование головной ветки "master" в "main"
git push -u origin main - синхронизация репозиториев на локальном компьютере с дистанционным сервером на GitHub ветку "main"
git push - просто в таком виде команда заливает все изменения с локального на дистанционный сервер.
git clone <скопированный адрес из Github-Code-HTTPS> - копирование или клонирование различных репозиториев на локальном сервере. 
В GitHub по вкладке <>Code/ Зелёная кнопка Code HTTPS - по ссылке либо скачать ZIP.
